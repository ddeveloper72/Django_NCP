Import MVC File from EU Central Terminology Server

Use this interface to import Master Value Set Catalogue (MVC) data from the EU Central Terminology Server. The MVC file contains healthcare value sets and their concepts for cross-border healthcare services.

What is an MVC File?

  • Master Value Set Catalogue - Contains standardized healthcare terminology
  • Multiple Sheets - Each sheet represents one value set (e.g., countries, languages, medical devices)
  • Concepts - Each value set contains multiple concepts with codes and descriptions
  • Multi-language - Supports translations for cross-border healthcare

Import Modes:

  • Sample Import - Import first 5 sheets only (recommended for testing)
  • Selective Import - Choose specific sheets to import
  • Full Import - Import all sheets (may take several minutes)
