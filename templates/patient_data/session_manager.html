💡 Development Tips
  • Session Duration: Development sessions last 7 days (vs 2 hours in production)
  • Auto-Creation: Missing sessions will be auto-created in development mode
  • CLI Management: Use python manage.py manage_sessions list for command line access
  • Debug Mode: Add ?debug=true to any patient URL for extra debugging info
🖥️ CLI Commands
# List all sessions
python manage.py manage_sessions list

# Create session for patient
python manage.py manage_sessions create --patient-id 2845672827

# Clean expired sessions
python manage.py manage_sessions clean

# Delete specific session
python manage.py manage_sessions delete --session-key abc123...
